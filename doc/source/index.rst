
Welcome to python-mobilitydb's documentation!
=============================================

``python-mobilitydb`` is a database adapter to access `MobilityDB <https://github.com/MobilityDB/MobilityDB>`_ from Python. It supports both the `psycopg2 <https://github.com/psycopg/psycopg2>`_ and the `asyncpg <https://github.com/MagicStack/asyncpg>`_ adapters for PostgreSQL and uses the `postgis <https://github.com/tilery/python-postgis>`_ adapter for PostGIS.
